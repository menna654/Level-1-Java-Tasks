Level 1 Java Tasks

This repository contains Java Level 1 internship tasks completed as part of my Java Development Internship at Codveda Technology.

 Projects Included
1. Number Guessing Game

A console-based Java game where the program generates a random number, and the user tries to guess it within a limited number of attempts.

 Features
* Generates a random number using Java Random class
* Takes user input from the console
* Gives feedback if the guess is too high or too low
* Limits the number of attempts
* Handles invalid inputs

 Concepts Used
* Random class
* Loops
* Conditional statements
* User input handling
* Basic error handling
---
 2. Factorial Using Recursion

A simple Java program that calculates the factorial of a number using recursion.

 Features
* Calculates factorial using a recursive method
* Handles factorial of 0
* Handles negative number input
* Displays the result in the console

 Concepts Used
* Recursion
* Methods
* Conditional statements
* Edge case handling

   How to Run

1. Clone the repository:
git clone https://github.com/menna654/Level-1-Java-Tasks.git
2. Open the project folder:
cd Level-1-Java-Tasks
3. Compile and run the Number Guessing Game:
javac NumberGuessingGame.java
java NumberGuessingGame
4. Compile and run the Factorial Recursion program:
javac FactorialRecursion.java
java FactorialRecursion
